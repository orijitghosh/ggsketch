# The available style presets

The valid values for
[`sketch_style()`](https://orijitghosh.github.io/ggsketch/reference/sketch_style.md).

## Usage

``` r
sketch_styles()
```

## Value

A character vector of style names.

## See also

Other sketch-style:
[`sketch_style()`](https://orijitghosh.github.io/ggsketch/reference/sketch_style.md)

## Examples

``` r
sketch_styles()
#> [1] "notebook"    "chalkboard"  "blueprint"   "field_notes" "graphite"   
```
